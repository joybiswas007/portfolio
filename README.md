# Joy Biswas - Portfolio

![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)

Welcome to my portfolio repository! This site showcases my skills, projects, and experience as a backend developer. It is hosted on GitHub Pages and built with HTML, CSS, and JavaScript.

## 🌐 Live Demo

Check out the live version of my portfolio:
**[https://joybiswas007.github.io/portfolio/](https://joybiswas007.github.io/portfolio/)**

## 🚀 Features

- **Clean and Minimal Design**: A simple, responsive, and professional layout.
- **Project Showcase**: Highlights my key projects with links to their repositories.
- **Education Section**: Details my academic background and achievements.
- **Contact Information**: Easy ways to get in touch with me.

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Hosting**: GitHub Pages
- **Version Control**: Git

## 🖥️ Running Locally

If you'd like to run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/joybiswas007/portfolio.git
   ```
2. **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```
3. **Open the project**
```bash
Open the index.html file in your browser.
```
